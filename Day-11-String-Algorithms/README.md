# 📖 What are String Algorithms?

String algorithms are techniques used to efficiently process, compare, search, and manipulate strings. They are commonly asked in coding interviews and help solve problems involving text processing.


---

# 🔹 Common Techniques

| Technique            | Used In 
|----------------------|---------------------------
| Two Pointers         | Subsequence, Compression 
| String Traversal     | Robot Return 
| Prefix Comparison    | Longest Common Prefix 
| String Matching      | Find First Occurrence 
| String Concatenation | Rotate String 

---

# 🔹 Time Complexity

| Operation               | Complexity 
|-------------------------|------------
| String Traversal        | O(n) 
| Prefix Comparison       | O(n × m) 
| String Matching (Naive) | O(n × m) 
| Subsequence             | O(n) 
| Rotate String           | O(n) 
| Compression             | O(n)
